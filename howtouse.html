HOW TO USE THE TAUSI PROGRAMMING LANGAUGE

TAUSI IS A SWAHILI PROGRAMMING LANGAUGE VARIOUS PROGRAMMING LANGUAGES ARE USED TO WRITE AND EXECUTE COMPUTER PROGRAMS SINCE TAUSI IS IN THE INITIAL STAGE OF ITS CREATION IS HAS VERY FEW SYNTAX AND KEYWORDS WHICH ARE AS FOLLOWS jumbe("neno") TO CREATE OR DECLARE VARIABLE OR ELEMENT WE USE tmksh REPRESENTING tumilikishi TMKSH namba=3 TO ALLOW THE PROGRAM TO PROMPT FOR A STANDARD INPUT FROM THE USER WE USE ingizo() e.g TMKSH jina=ingizo()
